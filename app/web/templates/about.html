Version {{ system_info.app_version }}

Description

SmartHomeLite is a lightweight, Python-based smart home hub designed to run on Android devices using Termux. It provides a local solution for managing and controlling smart home devices without relying on cloud services.

The project focuses on simplicity, extensibility, and privacy, allowing users to control their smart home devices through a local network without sending data to external servers.

Features

  • Bluetooth device discovery and control
  • RESTful API for device management
  • Responsive web dashboard
  • Optional MQTT integration
  • Optional voice control capabilities
  • Extensible architecture for future integrations
